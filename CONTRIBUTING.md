Thank you for your interest in Jarvis!

We appreciate issues and bug reports. At this time we are not accepting pull requests for code changes.

How to contribute:
- Search existing issues before filing a new one.
- Use the provided issue templates (Bug report, Feature request) and include steps to reproduce for bugs.
- Security issues: please see SECURITY.md for reporting instructions.

Code contributions (PRs) will be closed with a note referencing this policy.
